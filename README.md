# Repositorio de Aplicaciones, y de Recursos Educativos

En este repositorio se encuentran diversas aplicaciones `apps/` y recursos educativos `resources/`, desarrollados para apoyar la enseñanza de temas relacionados programación de sistemas con microcontroladores, sistemas embebidos, 
*Machine Learning* y otros tópicos afines. 

Este es un espacio dedicado a compartir herramientas por lo que siéntanse libres de explorar, utilizar y contribuir con sus propias aplicaciones o mejoras a las existentes.

Cada aplicación/recurso de aprendizaje está organizada en su propia carpeta dentro del directorio `apps/` para aplicaciones y `resources/` para recursos educativos, y cada una incluye su propia documentación y archivos necesarios para su ejecución/seguimiento.

-----

## Aplicaciones Disponibles
### [Syllabus and Note Generator](syllabus/README.md)

Una aplicación para crear y gestionar syllabus de cursos, actualmente tiene implementada la generación en formato LaTeX a partir de los datos obtenidos desde una base de datos SQLite o desde archivos CSV.

En desarrollo:
- Generación de calificaciones en formato CSV.
- Generar calificaciones parciales y finales automáticamente basadas en criterios definidos.
